# vite-chakraui-jianshu

vite-chakraui-jianshu

## 安装依赖

```dotnetcli
//  搭建 vite-react 开发环境
yarn create @vitejs/app vite-chakraui-jianshu --template react

// 安装 chakra-ui 库
cd /vite-chakraui-jianshu
yarn add @chakra-ui/react @emotion/react@^11 @emotion/styled@^11 framer-motion@^4

// 添加图标组件库
yarn add react-icons
```
